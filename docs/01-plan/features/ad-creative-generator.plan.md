# Plan: 광고 소재 생성 솔루션 (Ad Creative Generator)

> 작성일: 2026-02-05
> 상태: Draft
> Feature ID: ad-creative-generator

---

## 1. 개요 (Overview)

### 1.1 문제 정의
- 광고 소재 제작 시 "왜 이 이미지/카피를 선택했는지"에 대한 전략적 근거가 부족
- 비주얼과 카피 기획이 감에 의존하는 경우가 많음
- 플랫폼별(인스타그램, 틱톡, 쓰레드, 유튜브) 최적화된 소재 제작이 어려움

### 1.2 솔루션 목표
**AI 기반 광고 소재 생성 솔루션** - 분석 → 기획 → 생성의 전 과정을 지원

```
┌─────────────────────────────────────────────────────────────────┐
│                    Ad Creative Generator                        │
├─────────────────────────────────────────────────────────────────┤
│  [분석 단계]     →    [기획 단계]    →    [생성 단계]           │
│                                                                 │
│  • 타겟 분석         • 컨셉 도출         • 이미지 생성          │
│  • 경쟁사 분석       • 비주얼 기획       • 카피 생성            │
│  • 트렌드 분석       • 카피 기획         • 영상 생성(옵션)      │
│  • 플랫폼 분석       • 스토리보드        • 플랫폼별 리사이징    │
└─────────────────────────────────────────────────────────────────┘
```

## 2. 핵심 기능 (Core Features)

### 2.1 Phase 1: 분석 (Analysis)

| 기능 | 설명 | 우선순위 |
|------|------|----------|
| 타겟 오디언스 분석 | 연령, 성별, 관심사, 행동 패턴 분석 | P0 |
| 경쟁사 소재 분석 | 경쟁 브랜드 광고 소재 수집 및 분석 | P1 |
| 트렌드 분석 | 현재 광고 트렌드, 밈, 인기 포맷 분석 | P1 |
| 플랫폼별 특성 분석 | 각 플랫폼의 최적 포맷, 톤앤매너 분석 | P0 |

**분석 결과물:**
- 타겟 페르소나 문서
- 경쟁사 벤치마크 리포트
- 트렌드 인사이트 리포트
- 플랫폼별 가이드라인

### 2.2 Phase 2: 기획 (Planning)

| 기능 | 설명 | 우선순위 |
|------|------|----------|
| 크리에이티브 컨셉 도출 | 분석 기반 광고 컨셉 3-5개 제안 | P0 |
| 비주얼 기획 | 이미지 스타일, 구도, 컬러 기획 | P0 |
| 카피 기획 | 헤드라인, 바디카피, CTA 기획 | P0 |
| 스토리보드 | 영상 광고용 스토리보드 (옵션) | P2 |

**기획 산출물:**
- 크리에이티브 브리프
- 비주얼 무드보드
- 카피 가이드라인
- 스토리보드 (영상의 경우)

### 2.3 Phase 3: 생성 (Generation)

| 기능 | 설명 | 우선순위 |
|------|------|----------|
| AI 이미지 생성 | 기획 기반 광고 이미지 생성 | P0 |
| AI 카피 생성 | 플랫폼별 최적화된 카피 생성 | P0 |
| AI 영상 생성 | 이미지 기반 짧은 영상 생성 | P2 |
| 플랫폼별 리사이징 | 각 플랫폼 규격에 맞는 자동 리사이징 | P1 |

**생성 산출물:**
- 플랫폼별 이미지 에셋
- 플랫폼별 카피 에셋
- 영상 에셋 (옵션)

## 3. 지원 플랫폼 (Target Platforms)

| 플랫폼 | 이미지 규격 | 특성 |
|--------|-------------|------|
| Instagram Feed | 1080x1080, 1080x1350 | 비주얼 중심, 감성적 |
| Instagram Stories/Reels | 1080x1920 | 짧고 임팩트 있는 |
| TikTok | 1080x1920 | 트렌디, 밈 활용 |
| Threads | 1080x1080 | 텍스트+이미지 조합 |
| YouTube Shorts | 1080x1920 | 짧은 영상 중심 |
| YouTube Ads | 1920x1080, 1080x1080 | 다양한 포맷 |

## 4. 사용자 워크플로우 (User Workflow)

```
사용자 입력
    │
    ▼
┌─────────────────────────────────────────────────────────────┐

│ • 브랜드/제품 정보                                           │

    │
    ▼

    ▼

│ • 플랫폼별 자동 리사이징                                     │
│ → 광고 소재 패키지 다운로드                                  │

    │
    ▼
```

## 5. 기술 스택 제안 (Tech Stack)

### 5.1 프론트엔드
- **Framework**: Next.js 14+ (App Router)
- **UI**: Tailwind CSS + shadcn/ui
- **상태관리**: Zustand 또는 React Query

### 5.2 백엔드
- **API**: Next.js API Routes 또는 별도 서버
- **데이터베이스**: PostgreSQL (Supabase) 또는 bkend.ai
- **인증**: NextAuth.js 또는 bkend.ai Auth

### 5.3 AI 서비스

#### 5.3.1 이미지 생성: Nano Banana Pro (메인)

| 항목 | 내용 |
|------|------|
| **제공사** | Google DeepMind |
| **기반 모델** | Gemini 3 Pro Image Preview |
| **Model ID** | `gemini-3-pro-image-preview` |
| **접근 방법** | Google AI Studio / Vertex AI |
| **SDK** | REST API, Python SDK |

**핵심 기능:**
- 고해상도 네이티브 출력 (1K, 2K, 4K) - 업스케일링 불필요
- 선명한 텍스트 렌더링 (포스터, 로고용) - 다국어 지원
- 캐릭터/객체 일관성 유지 (멀티프레임)
- 자연어 기반 멀티스텝 이미지 편집

**API 가격 (2026 기준):**

| 해상도 | 공식 가격 | 서드파티 |
|--------|-----------|----------|
| 1K-2K | $0.134/이미지 | ~$0.02 (85%↓) |
| 4K | $0.24/이미지 | ~$0.12 (50%↓) |
| Batch API | 50% 할인 | - |

**무료 티어:**
- Google AI Studio: 일 50회 (월 ~1,500회)
- Google Cloud 신규: $300 크레딧 (90일, ~2,240개 4K 이미지)

**광고 소재에 적합한 이유:**
1. 4K 고해상도로 인쇄/디지털 모두 대응
2. 텍스트 렌더링으로 카피 포함 이미지 생성 가능
3. 캐릭터 일관성으로 시리즈 광고 제작 용이
4. Google 생태계와의 통합 (YouTube, Google Ads 등)

#### 5.3.2 텍스트 생성
- OpenAI GPT-4

#### 5.3.3 영상 생성

- **스토리지**: AWS S3 또는 Cloudflare R2

## 8. 리스크 및 대응 (Risks & Mitigation)

| 리스크 | 영향 | 대응 방안 |
|--------|------|-----------|
| AI 이미지 품질 불안정 | 높음 | 다중 AI 서비스 연동, 재생성 기능 |
| API 비용 증가 | 중간 | 사용량 제한, 캐싱, 프리미엄 플랜 |
| 저작권 이슈 | 높음 | 상업용 라이선스 확인, 면책조항 |
| 플랫폼 정책 변경 | 중간 | 모듈화 설계로 빠른 대응 |
